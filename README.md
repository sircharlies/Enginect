# SimCore Engine

> A lightweight 2D simulation engine built with TypeScript.

---

## 📖 About

SimCore is a personal project focused on learning and experimenting with:

- Engine Architecture
- Physics Simulation
- Rendering
- Mathematics
- Design Patterns
- Object-Oriented Programming
- Systems Architecture

Instead of creating isolated simulations, the goal is to build a reusable 2D engine capable of running different kinds of simulations.
